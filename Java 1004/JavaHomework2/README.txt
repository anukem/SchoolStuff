Problem1.java
	This program takes a year as input and returns the month and day that Easter will fall on that year.

Problem2.java
	This program takes two inputs. The first is the amount the customer owes, while the second is the amount given.
	It returns the amount of change due to customer in dollars, quarters, dimes, nickels, and pennies by dividing 
	and then modding the number until all that's left is pennies.

DegreeConverter.java
	This program takes two inputs, one to choose the conversion, and another for the number to do the conversion on.
	It does so by using the formulas as an algorithm and then prints the result.
